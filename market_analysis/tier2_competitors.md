# Tier 2 Competitors — Significant Niche Players

**Date:** February 2026

---

## 3.1 AstroSeek

- **URL:** https://www.astro-seek.com
- **Type:** Web-based
- **Positioning:** Technical astrologers; free, comprehensive, data-rich
- **Monthly Traffic:** ~3M/month

**Key Tools:**
- Free Natal Chart (many house system options — 15+ systems)
- Synastry Chart & Composite/Davison Chart
- Solar Return + Progressions
- **Astrocartography (completely free — competitors charge $30–$80)**
- Asteroid Calculator (100+ named asteroids)
- Fixed Star Calculator
- Arabic Parts / Lots Calculator
- Planetary Hours Calculator
- Eclipse Calendar
- Best Match Synastry Search
- Dignity Scorecard

**Unique Features:**
- Free astrocartography puts it ahead of Astro.com for this feature
- Most extensive free asteroid support of any platform
- Detailed dignity/debility analysis for traditional astrologers
- Active community forum

## 3.2 TimePassages (Astrograph Software)

- **URL:** https://www.astrograph.com / TimePassages mobile app
- **Type:** Desktop software + mobile app (iOS/Android)
- **Pricing:** Free basic app; Premium $9.99/year; Desktop $99–$499
- **Positioning:** Professional-grade, beautiful chart rendering, mobile convenience

**Key Tools:**
- Natal Chart (stunning jewel-tone wheel rendering — considered most beautiful)
- Transits & Progressions overlay
- Chart comparison
- Detailed written interpretations (premium)
- Planet positions tracking with push notifications

**Strengths:**
- Best-in-class chart wheel aesthetics (full color, ornate, gem-like quality)
- Offline capability (no internet required after data sync)
- Professional-grade calculations (Swiss Ephemeris)
- Intuitive mobile interface with bottom-drawer details
- Timeline/Gantt view for transit visualization

## 3.3 Pattern (App)

- **URL:** https://www.thepattern.com
- **Type:** Mobile only (iOS/Android)
- **Founded:** 2017
- **Revenue:** Subscription $19.99/year
- **Positioning:** Psychological personality tool using astrology; less "woo," more self-development

**Key Features:**
- Personality insights (no visible chart wheel — deliberately hidden)
- "Timing" feature for life cycles (transits rendered as personality phases)
- Relationship analysis
- Daily check-ins

**Strengths:**
- Excellent writing quality; psychological framing removes stigma for skeptics
- Celebrity following (Drake, Channing Tatum, Bella Hadid)
- Very high retention / daily engagement metrics
- Zero astrology jargon — accessible to non-astrologers

## 3.4 Sanctuary

- **URL:** https://sanctuaryworld.co
- **Type:** Mobile app (iOS/Android)
- **Revenue:** Subscription ($9.99/month) + live astrologer chat ($25–$50/session)
- **Positioning:** Premium spiritual wellness; live human astrologers + AI blend

**Key Features:**
- Daily personalized reading (AI-generated from chart)
- Live 1:1 astrologer sessions via chat/video
- Birth chart
- Weekly forecasts
- Ritual suggestions and moon phase guidance

## 3.5 Chani App (CHANI)

- **URL:** https://chaninicholas.com/app
- **Type:** Mobile app (iOS/Android)
- **Revenue:** $19.99/year subscription
- **Positioning:** Queer-affirming, feminist, social justice-oriented astrology

**Key Features:**
- Weekly/monthly audio readings (CHANI Nicholas's voice — intimate and personal)
- Ritual prompts tied to lunar calendar
- New + Full Moon guides with journaling prompts
- Chart basics

**Strengths:** Strong author brand, fiercely loyal progressive audience, audio-first differentiation sets it apart from all competitors
