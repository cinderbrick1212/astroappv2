## Recommended Demo Video

Record a 60–90 second screen recording covering:
1. Launch → Home screen (horoscope + streak)
2. Tap **Kundli** → enter birth details → view chart
3. Tap **Dasha** → mahadasha timeline
4. Tap **Panchang** → today's Tithi / Nakshatra
5. Tap **Tools** → browse all 16 tools
6. Profile screen → language switch (English ↔ Hindi)

Save the recording as `demo.mp4` in this directory, or upload to YouTube/Loom and link from the root README.
